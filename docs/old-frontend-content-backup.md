## API Endpoints

### Base URL
- **Development:** http://localhost:4200/api
- **Environment Variable:** NEXT_PUBLIC_API_URL

### Endpoints

#### 1. Join Community
- **Method:** POST
- **Path:** `/users/join/community`
- **Body:**
  ```json
  {
    "email": "string (required)",
    "name": "string (optional)",
    "communities": "string[] (required)",
    "githubHandle": "string (optional)"
  }
  ```

#### 2. Submit Mentor Application
- **Method:** POST
- **Path:** `/users/mentor/apply`
- **Body:**
  ```json
  {
    "email": "string (required)",
    "name": "string (required)",
    "expertise": "string (optional)",
    "githubHandle": "string (optional)",
    "motivation": "string (optional)"
  }
  ```

#### 3. Newsletter Subscription
- **Method:** POST
- **Path:** `/users/newsletter/subscribe`
- **Body:**
  ```json
  {
    "email": "string (required)"
  }
  ```

### API Client Configuration
- **Default Timeout:** 10 seconds (10000ms)
- **Headers:**
  - Content-Type: application/json
- **Error Handling:**
  - Network errors
  - Timeout errors (408)
  - API errors with status codes
